## 车流量项目
>功能：通过socket实现网关设备数据的接收，提供webservice，即数据格式为xml的API。

>接收程序：
* 数据包格式参照文档：交通信息采集系统接口说明1
* 因为CrossID字段是特殊的8位连续数字且每包字节流中有且只有一个CrossID，所以以此作为判断包开头的条件
* 一个包最多包含32个地磁的信息，故设置一个包拼接的最大长度2000
* 只有接收的数据长度刚好等于预测接收数据长度，才会调用处理数据块
* 只有数据长度和校验码都检查正确，才会去存数据
* PacketType决定是否返回时间戳和存储：1代表只存，128代表只回时间，129代表既存又回时间

>接口程序
* 模型js文件定义方法，返回的是JSON字符串
* JSON到XML的转化过程在router进行
